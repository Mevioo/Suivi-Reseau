Bienvenue dans la documentation de Suivi-Reseau
===============================================

Bienvenue dans le projet **Suivi-Réseau**, un outil de scan réseau asynchrone
développé en Python. Ce projet permet d'identifier les adresses IP actives
dans un réseau local à partir d'une plage CIDR ou d'un fichier d'adresses IP.
